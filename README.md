# Housing Price Prediction

## Project Overview
This project focuses on predicting housing prices using Machine Learning algorithms based on property features such as area, bedrooms, bathrooms, parking, furnishing status, and location-related facilities.  
The main goal is to build a regression model that can estimate house prices accurately.

---

## Workflow

1. **Data Collection**  
   Imported Housing.csv dataset.

2. **Data Preprocessing**  
   - Checked missing values  
   - Removed duplicates  
   - Encoded categorical columns

3. **Exploratory Data Analysis**  
   - Histograms  
   - Boxplots  
   - Heatmap  
   - Feature relationships

4. **Feature Selection**  
   Selected important variables affecting price.

5. **Model Training**  
   Applied regression algorithms:
   - Linear Regression  
   - Decision Tree Regressor  
   - Random Forest Regressor

6. **Model Evaluation**  
   Used:
   - R² Score  
   - Mean Squared Error

7. **Prediction**  
   Predicted housing prices using trained model.
